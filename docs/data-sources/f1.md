## Series Overview

The FIA Formula 1 World Championship. 24 rounds per season (2026), weekend format includes FP1/FP2/FP3, Qualifying, and Race. Some weekends replace FP2/FP3 with Sprint Qualifying + Sprint Race. Best-documented series with multiple free public APIs.

## APIs & Sources Researched

### Source 1 тАФ Jolpica API (primary)

- **URL:** `https://api.jolpi.ca/ergast/f1/{year}.json`
- **Auth required:** No
- **Status:** Working
- **Data quality:** Full calendar with circuits, GPS coords, session times (FP1/FP2/FP3/Quali/Sprint). Ergast successor тАФ actively maintained.
- **Rate limiting:** None observed
- **Sample response:**
```json
{
  "MRData": {
    "RaceTable": {
      "Races": [
        {
          "round": "1",
          "raceName": "Bahrain Grand Prix",
          "Circuit": {
            "circuitId": "bahrain",
            "circuitName": "Bahrain International Circuit",
            "Location": { "lat": "26.0325", "long": "50.5106", "country": "Bahrain" }
          },
          "date": "2026-03-15",
          "FirstPractice": { "date": "2026-03-13", "time": "11:30:00Z" },
          "Qualifying": { "date": "2026-03-14", "time": "15:00:00Z" },
          "Race": { "date": "2026-03-15", "time": "15:00:00Z" }
        }
      ]
    }
  }
}
```
- **Notes:** Country field is full name (e.g. "Bahrain"), not ISO code. Must derive `countryCode` separately or from circuit data.

### Source 2 тАФ OpenF1 API (supplementary)

- **URL:** `https://api.openf1.org/v1/sessions?year=2025`
- **Auth required:** No
- **Status:** Working тАФ 123 sessions for 2025
- **Data quality:** Very granular session-level data including lap-by-lap telemetry. More than needed for calendar purposes.
- **Rate limiting:** None observed
- **Notes:** Better suited for live data and detailed session info than calendar building. Used as fallback.

---

## Decision

**Chosen source:** Jolpica API (primary) + OpenF1 as fallback
**Integration path:** API fetcher (`pipeline/fetchers/f1.py`)

**Rationale:**
Jolpica is the direct successor to the Ergast API, which was the industry standard for F1 data for 15+ years. It's free, no auth, well-structured, and includes all session times in UTC. OpenF1 supplements with live/real-time data if needed.

**Trade-offs accepted:**
- Country names need to be mapped to ISO alpha-2 codes manually (Jolpica returns full names)
- No broadcast info in the API тАФ needs manual curation
